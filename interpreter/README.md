# #Interpreter

对于一种语言，模式定义了它的语法的表示以及
使用表示来解释语言中的句子的解释器。
解释器模式具有行为目的，并应用于类。

###何时使用

*当语法简单时(如果语法复杂，有更好的选择)
效率不是一个关键问题
