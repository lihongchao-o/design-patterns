# #Decorator

动态地向对象附加额外的责任。
修饰符
为扩展功能提供一种灵活的替代方法。
该模式具有结构化目的，并适用于对象。

###何时使用

动态透明地向单个对象添加责任，也就是说，不影响其他对象
*可以撤销的责任
*当通过子类化扩展不切实际时
